# Sta Maria Enrollment System

A comprehensive school management and enrollment platform designed for administrators, teachers, and parents.

## Features

- **Admin Dashboard**: Manage school operations, students, and staff.
- **Teacher Portal**: Track student progress, attendance, and grades.
- **Parent Portal**: Monitor child's enrollment status and school updates.
- **Modern UI**: Built with React, TypeScript, and Tailwind CSS v4.

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Tech Stack

- **Framework**: React 18
- **Styling**: Tailwind CSS v4
- **Routing**: React Router Dom
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Components**: Radix UI Primitives
